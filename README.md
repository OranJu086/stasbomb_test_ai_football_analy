# stasbomb_test_ai_football_analy

本仓库展示了一个使用 Python 可视化技术实现足球比赛进程可视化的项目。通过分析比赛事件数据，项目能够直观呈现球员传球网络、关键进攻动作以及比赛整体战术走向。

## 功能特点

- **比赛事件可视化**：展示每场比赛中球员的传球、射门等事件。

- **传球网络分析**：通过图形化方式呈现球员间的传球关系。

- **战术动态呈现**：利用 Python 可视化库，将比赛进程以动画或图表形式直观展示。

- **数据驱动**：基于真实比赛数据进行分析，可扩展至多场比赛。

## 技术栈

- Python 3.x

- Matplotlib / Seaborn / NetworkX 等可视化库

- JSON 数据解析
